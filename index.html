Web

  • What is HTTP?

    • HTTP is how browsers and servers communicate

  • What is a URL?

    • a URL is how you access a website... it has protocol, hostname, port, response and query

  • What is DNS?

    • DNS is Domain Name System. it turns the hostname into an IP address.. like a server phone book

  • What is a query string?

    • a query string provides extra info such as search items or form info

  • What are two HTTP verbs?

    • Get
    • Post

  • How are they different?

    • Get requests are without side effects and arguments are normally passed in query string

    • Post requests are with side effects and arguments are normally sent as body

  • What is an HTTP request?

    • An HTTP request is when you point your browser to a webpage on a server. You need a method, HTTP protocol version, resourse URL, and optionally, headers.

  • What is an HTTP response?

    • an HTTP response is when the server responds with the exact HTML text for said page. It has an HTTP protocol version, reponse status code, and optionally, headers.

  • What is an HTTP header?

    • an HTTP header is for specifying how and what you want the page to be

  • What are some request headers?

    • request headers can be the hostname, the date, language, cookies, etc..

  • What are some response headers?

    • response headers can be content type, date/time, cookies, cached info, etc..

  • What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?

    • DNS turns the hostname into an IP address and then connects to the server. Default connects to port 80 and uses HTTP protocol asking for '/some/page.html'
